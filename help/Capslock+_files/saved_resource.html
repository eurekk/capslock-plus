                                                                                              完全可以自己添加，代码都在lib目录里，
                                                                                              在当前行后插入一行的代码是：
                                                                                              keyFunc_enterWherever(){
                                                                                              SendInput,{End}{Enter}
                                                                                              Return
                                                                                              }
                                                                                              那么你可以添加一个函数
                                                                                              keyFunc_enterWhereverbefore(){
                                                                                              SendInput,{Home}{Enter}{Up}
                                                                                              Return
                                                                                              }
                                                                                              然后绑定一个快捷键，比如你说的ctrl+shift+enter就可以了。
